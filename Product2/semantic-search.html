Semantic CMDB Search Tool

A prototype that enables natural language search across configuration and dependency data to surface relevant systems, relationships, and evidence.

Executive summary

The Semantic CMDB Search Tool is an AI-powered prototype that transforms complex ServiceNow Configuration Management Database (CMDB) data into actionable insights. Users can ask questions in plain English and instantly locate configuration items, dependencies, and key metadata—no technical query skills required

Inspired by my experience as an Application Portfolio Manager, the tool addresses the inefficiency of searching large, distributed CMDB datasets. Traditional interfaces require specialized knowledge and manual filtering, slowing discovery for both IT teams and business stakeholders.

By bridging structured enterprise data with natural language search, the prototype modernizes workflows while maintaining data integrity and privacy.

Overview

Enterprise CMDBs can contain thousands of applications, infrastructure components, and service records. Finding the right information quickly is often challenging. This tool simplifies discovery, enabling teams to search, explore, and act on CMDB data efficiently.

Key features

  • Natural Language Search: Ask questions in plain English and receive accurate results.
  • Semantic Understanding: AI interprets meaning and context beyond keyword matching.
  • Private & Secure: Sensitive data stays within the enterprise environment.
  • Fast Retrieval: High-speed search reduces discovery time from minutes to seconds.
  • Scalable Design: Easily extendable to additional data sources or systems.

Target users

  • Application Portfolio Managers seeking faster visibility into system and service data.
  • ServiceNow Administrators looking to simplify CMDB search and analysis.
  • IT Operations and Service Teams needing rapid insights during incidents or audits.
  • Business Stakeholders who require access to system ownership and dependencies without technical barriers.

Business value

  • Efficiency: Accelerates data discovery and decision-making.
  • Accessibility: Makes complex IT data usable for all stakeholders.
  • Faster Decisions: Supports incident response, application reviews, and audit.
  • Governance & Security: Maintains integrity and privacy of enterprise data.

Technology overview

  • AI-Powered Semantic Search: Uses OpenAI embeddings to understand user queries.
  • Backend Services: Orchestrates queries and delivers results in real time.
  • Local Vector Database: Ensures fast, private, and scalable data retrieval.
  • Streamlined Web Interface: Interactive and easy-to-use for non-technical users.

Note: The prototype connects to a ServiceNow Personal Developer Instance (PDI) for realistic testing while keeping live enterprise data secure.

MVP scope

  • Connect to ServiceNow PDI to retrieve CMDB records.
  • Generate and store embeddings in Chroma vector database.
  • Process user queries through OpenAI embeddings.
  • Perform semantic search via Chroma and return relevant metadata.
  • Display results in Streamlit with key attributes.

Roadmap / Future Enhancements

  • Simplified deployment for broader adoption.
  • Role-based user access for secure multi-user environments.
  • Automated refresh of semantic data for dynamic CMDB updates.
  • Conversational, chatbot-style search interface.
  • Integration with additional IT and business data sources.
